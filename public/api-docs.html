EngVox API Documentation

Backend API v4.0.1 — englishengineer-production.up.railway.app

Health

GET /api/health Public
Returns system health status including AI and Stripe configuration.
Response: { ok: true, version: "4.0.1", aiConfigured: true, stripeConfigured: true }

AI Routes

POST /api/ai/coach Auth Required
AI coaching endpoint for writing feedback and language improvement.
Body: { prompt: string, mode: "writing"|"speaking" }
POST /api/ai/writing-review Auth Required
AI-powered writing review for engineering documents.
Body: { text: string, type: "rfi"|"ncr"|"email"|"report" }
POST /api/ai/assessment-feedback Auth Required
Assessment feedback for reading/listening tasks.

Billing

POST /api/billing/checkout Auth Required
Create Stripe checkout session for plan upgrade.
Body: { priceId: string, successUrl: string, cancelUrl: string }
GET /api/billing/subscription Auth Required
Get current subscription status.
POST /api/billing/portal Auth Required
Create Stripe customer portal session.

Vocabulary

GET /api/vocabulary/lookup Auth Required
Look up vocabulary terms by level and domain.
Query: level=A1|A2|B1|B2|C1|C2 & domain=string

Workspace

GET /api/workspace Auth Required
List user workspaces.
POST /api/workspace Auth Required
Create new workspace.
Body: { name: string }

Webhooks

POST /api/webhooks/stripe Stripe
Stripe webhook endpoint for billing events.
