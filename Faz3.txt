(özellikle Kademe 27 — üç kez ertelenen seed lazy-load — bu kez doğrulandı).

## 📋 Faz 3 Kademeleri (51-60)

### [x] Kademe 51: CI'ye Backend Typecheck Gate Ekleme
* **Açıklama:** `.github/workflows/ci.yml` şu an sadece
  `npm run backend:test` çalıştırıyor, backend `typecheck`'i hiç
  kontrol etmiyor. Bu yüzden 9 tip hatası fark edilmeden kaldı.
* **İşlem:** `ci.yml`'e backend testinden önce şu adımı ekle:
  ```yaml
  - name: Backend Typecheck
    run: npm --prefix backend run typecheck
  ```
* **ZORUNLU DOĞRULAMA:**
  - `ci.yml`'deki diff'i yapıştır.
  - Bu adımı eklendikten hemen sonra lokal olarak
    `npm --prefix backend run typecheck` çalıştır — şu an **fail etmesi
    bekleniyor** (9 hata var, henüz düzeltilmedi). Bu beklenen bir
    kırmızı sonuç, kademe 52'de düzeltilecek. Çıktıyı
    `docs/faz3-evidence/kademe51.txt`'e kaydet.
* **Durum:** Beklemede. *(Kullanıcı onayı gerekli — CI davranışı değişiyor.)*

### [x] Kademe 52: Backend'deki 9 TypeScript Hatasının Düzeltilmesi
* **Açıklama:** Kademe 43'teki (`any` temizliği) dikkatsiz değişiklik
  şu hataları bıraktı:
  - `src/middleware/rbac.middleware.ts` — TS2345
  - `src/rate-limit.ts` — TS2304: Cannot find name 'RateBucket'
  - `src/subscription-repository.ts` — TS2345
  - `src/supabase-audit-log-repository.ts` — TS2322
  - `src/vocabulary-routes.ts` — TS2352
  - `src/workspace.ts` — TS2345 (×3), TS2464
* **İşlem:** Her hatayı `any` ile "susturmak" yerine gerçek, doğru tiple
  düzelt. Özellikle `rate-limit.ts`'deki `RateBucket` tanımsız hatası
  muhtemelen eksik bir `import` — önce onu kontrol et, tip tamamen
  silinmiş/yanlış refactor edilmiş olabilir.
* **ZORUNLU DOĞRULAMA:**
  - `npm --prefix backend run typecheck` → **0 hata**, tam çıktıyı
    `docs/faz3-evidence/kademe52.txt`'e kaydet.
  - `npm --prefix backend test` → hâlâ 152/152 (veya üstü) geçmeli,
    hiçbir testin bu düzeltmeyle kırılmadığını göster.
* **Durum:** Beklemede.

### [x] Kademe 53: Frontend'deki Yeni axios/google-tts-api Açığı
* **Açıklama:** `npm audit` şu an `google-tts-api` → eski `axios`
  zincirinden **2 yüksek seviye açık** gösteriyor (Faz 2'de yoktu,
  muhtemelen ses/TTS özelliği eklenirken gelmiş).
* **İşlem:** `npm audit fix` dene; breaking change gerekiyorsa
  `google-tts-api`'yi güncel bir alternatifle değiştirmeyi değerlendir
  (örn. resmi bir TTS SDK'sı veya sürüm sabitleme). Sadece
  `--force` ile körlemesine güncelleme yapma — güncelleme sonrası
  TTS özelliğinin gerçekten çalıştığını manuel test et.
* **ZORUNLU DOĞRULAMA:**
  - `npm audit` çıktısı → axios/google-tts-api açığı görünmemeli,
    `docs/faz3-evidence/kademe53.txt`'e kaydet.
  - TTS özelliğinin (varsa ilgili sayfa/bileşen) hâlâ çalıştığına dair
    kısa bir not (manuel test sonucu).
* **Durum:** Beklemede.
