2. Adım: Hangi Dosyada Ne Değişecek?
İşi kendin veya AI ile yaparken odaklanacağın ana dosya genellikle styles/globals.css (veya projenin ana CSS dosyası) olacaktır.

Yapılması gereken teknik değişiklik şudur:

/* 1. ANA CSS DOSYASI (Örn: globals.css) */

:root {
  /* Gündüz Modu Değişkenleri */
  --background: #ffffff;
  --foreground: #171717;
  --card-bg: #f9f9f9;
  --primary-blue: #0070f3;
  --text-muted: #666666;
}

[data-theme='dark'] {
  /* Gece Modu Değişkenleri - Patlamayı önleyen tonlar */
  --background: #121212;     /* Saf siyah değil, gözü yormaz */
  --foreground: #ededed;     /* Saf beyaz değil, parlama yapmaz */
  --card-bg: #1e1e1e;        /* Derinlik hissi verir */
  --primary-blue: #3291ff;   /* Karanlıkta daha iyi görünen mavi */
  --text-muted: #a0a0a0;     /* İkincil yazılar için yumuşak gri */
}

/* Tüm sitede renkleri böyle kullanmalısın: */
body {
  background-color: var(--background);
  color: var(--foreground);
}
