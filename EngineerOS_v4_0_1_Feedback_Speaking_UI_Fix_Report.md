# EngineerOS v4.0.1 Feedback and Speaking UI Fix Report

- Feedback opens as an accessible dialog and closes through Cancel, X, Escape, backdrop click, route change or successful submission.
- Cancel never submits or changes learning state.
- Speaking mission selection and evaluation results now use explicit white/slate/soft-sky surfaces.
- The shared evaluation result includes a stable test identifier and contains no black result surface.
- Unit and browser coverage protect modal closure and light result behavior.
